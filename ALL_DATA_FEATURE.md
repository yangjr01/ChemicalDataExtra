# 所有文献数据汇总功能实现总结

## 📋 功能概述

在文献管理页面 `/chemical/literature` 添加了"数据汇总"入口，新页面 `/chemical/alldata` 可查看所有文献的材料、工艺、表征数据汇总。

## 🗂️ 创建的文件

### 后端文件
- **`server/endpoints/chemical/alldata.js`** - 新 API 端点
  - `GET /api/chemical/alldata` - 获取所有数据（支持筛选）
  - `GET /api/chemical/alldata/export?format=json|csv` - 导出数据

### 前端文件
- **`frontend/src/pages/Chemical/AllDataViewer.jsx`** - 数据汇总页面
- **`frontend/src/utils/paths.js`** - 添加 `paths.chemical.allData()` 路径
- **`frontend/src/main.jsx`** - 添加 `/chemical/alldata` 路由
- **`frontend/src/pages/Chemical/LiteratureManage.jsx`** - 添加"数据汇总"按钮

## 🎨 页面功能

### 1️⃣ 统计卡片
- 📊 **材料总数** - 显示所有文献的材料数量
- 🔧 **工艺总数** - 显示所有文献的工艺步骤数量
- 🔬 **表征总数** - 显示所有文献的表征数据数量

### 2️⃣ 筛选功能
- **文献筛选** - 下拉选择特定文献
- **关键词搜索** - 搜索文献标题
- **应用/清除筛选** - 快速筛选数据

### 3️⃣ 三个数据表格

#### 材料表
可显示字段（可自定义）：
- ✅ 文献标题
- ✅ 材料名称
- ✅ 化学式
- ✅ 材料类型
- ⭕ 组成成分（JSON）
- ⭕ 材料属性（JSON）
- ⭕ 备注

#### 工艺表
可显示字段（可自定义）：
- ✅ 文献标题
- ✅ 工艺名称
- ✅ 步骤序号
- ✅ 工艺描述
- ⭕ 工艺条件（JSON）
- ⭕ 工艺参数
- ⭕ 备注

#### 表征表
可显示字段（可自定义）：
- ✅ 文献标题
- ✅ 表征技术
- ✅ 测试材料
- ⭕ 关联工艺
- ⭕ 测试条件（JSON）
- ⭕ 测试结果（JSON）
- ⭕ 备注

**✅ = 默认显示，⭕ = 可选显示**

### 4️⃣ 自定义字段
- 点击"自定义字段"按钮
- 可独立配置三个表格的显示字段
- 实时生效

### 5️⃣ 导出功能
- **导出 JSON** - 导出完整数据结构
- **导出 CSV** - 导出为表格格式

### 6️⃣ 快速导航
- 每行数据都有"查看文献"按钮
- 点击可跳转到对应文献的数据查看页面

## 📊 数据流

```
用户点击"数据汇总"
    ↓
AllDataViewer 页面加载
    ↓
调用 GET /api/chemical/alldata
    ↓
后端查询三个表：
  - chemical_materials
  - chemical_processes
  - chemical_characterizations
    ↓
关联 chemical_articles 获取文献标题
    ↓
返回聚合数据
    ↓
前端渲染三个表格
```

## 🔧 API 接口

### 获取所有数据
```http
GET /api/chemical/alldata
Headers: Authorization: Bearer <token>

Query Parameters:
  - articleId: 按文献ID筛选
  - materialName: 按材料名称筛选
  - processName: 按工艺名称筛选
  - technique: 按表征技术筛选
  - search: 搜索文献标题

Response:
{
  "success": true,
  "data": {
    "materials": [...],
    "processes": [...],
    "characterizations": [...],
    "totalArticles": 10
  }
}
```

### 导出数据
```http
GET /api/chemical/alldata/export?format=json|csv
Headers: Authorization: Bearer <token>

Response: 文件下载 (chemical_data.json 或 chemical_data.csv)
```

## 🚀 使用说明

1. **访问数据汇总页面**
   - 在文献管理页面点击"数据汇总"按钮
   - 或直接访问 `/chemical/alldata`

2. **查看数据**
   - 点击三个标签页切换查看材料、工艺、表征数据
   - 查看统计卡片了解数据总量

3. **筛选数据**
   - 选择特定文献查看该文献的数据
   - 使用搜索框搜索文献标题

4. **自定义显示**
   - 点击"自定义字段"按钮
   - 勾选/取消勾选要显示的字段

5. **导出数据**
   - 点击"导出JSON"导出完整数据
   - 点击"导出CSV"导出表格格式

6. **查看详情**
   - 点击表格行末的"查看文献"按钮
   - 跳转到对应文献的数据查看页面

## ✅ 技术特点

- **单页应用** - 无需跳转，三个表格在同一页面
- **实时筛选** - 支持多条件组合筛选
- **字段自定义** - 每个表格可独立配置显示字段
- **数据关联** - 自动关联文献标题，显示友好
- **快速导航** - 一键跳转到文献详情
- **导出友好** - 支持 JSON 和 CSV 两种格式

## 📝 数据库查询说明

页面加载时会执行以下查询：

1. 查询所有文献（获取 ID 和标题映射）
2. 查询所有材料数据（带文献标题）
3. 查询所有工艺数据（带文献标题）
4. 查询所有表征数据（带文献标题）

所有查询通过 `articleId` 关联，确保数据一致性。
