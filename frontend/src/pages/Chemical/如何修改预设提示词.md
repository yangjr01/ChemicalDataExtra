# 如何修改预设提示词

## 快速开始

### 方法一：直接修改文件（最简单）

1. **找到提示词文件**
   ```
   anything-llm/server/storage/prompts/chemical/
   ```

2. **编辑对应文件**
   - `material_structured_extraction.txt` - 材料信息提取
   - `process_structured_extraction.txt` - 工艺参数提取
   - `characterization_structured_extraction.txt` - 表征数据提取
   - `process_flow_pre_extraction.txt` - 工艺流程预提取
   - `characterization_pre_extraction.txt` - 表征信息预提取

3. **保存并重启后端服务**

---

### 方法二：使用管理页面（推荐）

1. **访问管理页面**
   ```
   http://localhost:3000/chemical/prompts
   ```

2. **功能**
   - ✅ 查看和编辑提示词
   - ✅ 创建自定义提示词
   - ✅ 从预设模板创建
   - ✅ 激活/停用提示词
   - ✅ 删除提示词

---

### 方法三：通过 API

```bash
# 获取提示词列表
curl http://localhost:3001/api/chemical/prompts/list

# 创建提示词
curl -X POST http://localhost:3001/api/chemical/prompts/custom \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my_extraction",
    "title": "我的提取",
    "content": "提示词内容...",
    "category": "custom"
  }'

# 更新提示词
curl -X PUT http://localhost:3001/api/chemical/prompts/1 \
  -H "Content-Type: application/json" \
  -d '{"content": "新内容..."}'
```

---

## 提示词格式示例

```txt
你是一个专业的材料科学文献分析助手。请从给定的文献内容中提取材料的结构化信息。

请按以下 JSON 格式输出：
[
  {
    "name": "材料名称",
    "formula": "化学式/分子式",
    "type": "材料类型",
    "composition": {
      "elements": ["元素 1", "元素 2"],
      "ratios": "元素比例",
      "dopants": ["掺杂元素"]
    },
    "properties": {
      "crystalStructure": "晶体结构",
      "particleSize": "粒径",
      "surfaceArea": "比表面积"
    },
    "role": "在研究中的作用",
    "preparationMethod": "制备方法简述"
  }
]

请仔细识别文献中提到的所有材料，包括：
1. 原料/前驱体
2. 中间产物
3. 最终产物
4. 对比材料

如果某些信息文献中未提及，请标记为 null。请用中文回答。
```

---

## 注意事项

1. **修改文件后需要重启后端服务**
2. **JSON 格式必须正确**，否则无法解析
3. **使用中文回答**，确保输出语言一致
4. **测试验证**：修改后用相同文献测试效果
