# 表征数据图片提取功能

## 📋 功能概述

在提取流程页面的"第2部分：图片数据提取"中，新增**表征数据选择器**功能。允许用户：

1. 选择数据库中已有的表征数据
2. 在PDF图片中标定数据点
3. 将提取的数值自动填充到表征数据中

## 🎯 使用流程

### 1. 准备工作

- 确保已在数据查看页面保存了表征数据
- 进入提取流程页面 `/chemical/extract/:articleId`
- 切换到"第2部分：图片数据提取"

### 2. 选择表征数据

```
┌─────────────────────────────────────────────────────────────┐
│ 📊 数据库表征数据                                          │
│ 选择已保存在数据库中的表征数据，从图片中提取具体数值并填充  │
│                                                             │
│ [选择要填充的表征数据 ▼]  [从图片提取数据值]               │
│                                                             │
│ 已有提取值的表征：                                          │
│ ✓ XRD: 42.5  ✓ SEM: 15.2  ✓ BET: 850                      │
└─────────────────────────────────────────────────────────────┘
```

### 3. 提取数据值

1. **上传或截图** - 从PDF截取表征图表
2. **标定坐标轴** - 设置X轴和Y轴的标定点
3. **标定数据点** - 在曲线上点击标定具体数值点
4. **选择表征** - 从下拉框选择要填充的表征项
5. **点击"从图片提取数据值"** - 打开提取对话框

### 4. 填充数据

```
┌────────────────────────────┐
│ 提取数据值到表征           │
│                            │
│ 数据值: [42.5    ]         │
│ 单位:   [dB      ]         │
│                            │
│ [取消]  [保存到表征]       │
└────────────────────────────┘
```

点击"保存到表征"后：
- 数据值自动保存到表征的 results 字段
- 显示成功提示
- 表征选择器下方显示已提取的表征列表

## 🔧 技术实现

### 前端组件

**ImageCalibrationPanel.jsx**

新增功能：
- `CharacterizationSelector` - 表征选择器组件
- `loadCharacterizations()` - 加载数据库表征数据
- `saveValueToCharacterization()` - 保存提取值到表征
- `getCharExtractedValue()` - 获取已提取值显示

**表征选择器 (ChartSelector.jsx)**

```javascript
export const CharacterizationSelector = ({ 
  value, 
  onChange, 
  label = '选择表征数据',
  characterizations = []
}) => {
  // 从数据库加载表征列表
  // 显示技术名称和材料名称
}
```

### 后端API

**GET /api/chemical/data/:articleId**
- 获取文献的完整数据，包括表征列表

**PUT /api/chemical/data/characterization/:id**
- 更新表征数据
- 将提取的数值保存到 results 字段

数据格式：
```json
{
  "technique": "XRD",
  "conditions": "{...}",
  "results": {
    "imageExtractedValue": "42.5",
    "imageExtractedUnit": "dB",
    "extractionTimestamp": "2024-03-14T10:30:00.000Z"
  }
}
```

## 💡 优势

1. **关联数据库** - 直接从已保存的表征数据中选择
2. **可视化提取** - 在PDF图片上直观标定数据点
3. **自动填充** - 一键将提取值保存到表征
4. **可追溯** - 记录提取时间戳，便于追溯
5. **单位支持** - 支持自定义单位

## ⚠️ 注意事项

1. **需要先保存表征** - 必须在数据查看页面先保存表征数据
2. **坐标标定** - 提取前需完成坐标轴标定
3. **数据点标定** - 至少标定一个数据点才能提取
4. **单位规范** - 建议统一使用标准单位

## 🔗 相关功能

- 数据查看页面 `/chemical/data/:articleId` - 保存表征数据
- 提取流程页面 `/chemical/extract/:articleId` - 图片数据提取
- 坐标标定功能 - 在图片上标定坐标点
- 曲线提取功能 - 从图片提取数据曲线
